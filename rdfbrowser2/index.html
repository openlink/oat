        What's This?

        ODE Linked Data Web Search

        Welcome to OpenLink Data Explorer's search. There are two ways to proceed from here:

        • Searching for a text pattern in Virtuoso triple store
        • Dereferencing an URI - the ODE will use the RDFizer proxy to retrieve RDF data into ODE's RDF Cache and display it in the ODE Navigator

        Text Patterns

        The patterns can be quoted or non-quoted strings with wildcards and connectives AND, OR, AND NOT. For example "virtuoso AND NOT violin"
