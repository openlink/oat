OAT.Anchor.assign(anchor, optionsObject)

Modifies anchod element, so it acts as Enhanced Anchor: when activated, it displays some additional data in lookup bubble. The lookup bubble is an instance of OAT.Window object. Options:

  • href - string, url of requested data. when false, anchor inherits this value from href= attribute
  • connection - OAT.Connection object
  • datasource - OAT.DataSource object
  • imagePath - path for images used for lookup bubble
  • result - "grid", "timeline" or "form"
  • activation - "hover" or "click"
  • width, height - dimensions of lookup window
  • newHref - new 'href=' attribute of the anchor, defaults to 'javascript:void(0)'
